Luddesdowne Cricket Club

Luddesdowne Cricket Club

Stack: HTML, CSS, Javascript, Python (Django Framework), SQL-Lite

My cricket club was in great need of an update to the website, so I volunteered to step in and build one. Collaborating with a friend, we created a clean and simple design that allowed me to add some nice touches.

Using the Django Framework, I have been able to add a few features to the site. It currently has an API connection with Play Cricket and grabs the fixtures for the coming season, stores and then dynamically displays them on the front end.

Additionally, I have enabled our social team to be able to login and enter any social days and times. The site will then update the database with these and populate the front end dynamically. Unfortunately, as it stands, it looks like the social secretary has forgotten to update this!

I am also responsible for organising and running our annual fantasy cricket club - this is currently on excel. However, as a future project, I have been planning to incorporate this within the Luddesdowne site.

Xcom Talent Calculator and Memory Game

Xcom Talent Calculator and Memory Game

As part of my Diploma, I built this website to utilise and showcase Javascript and create two seperate areas.

The first feature is a talent calculator for a game. By using Javascipt, it allows the user to pre plan how they want to build the soldiers. I also teamed up with a content creator, who was happy for me to include his videos explaining the pros and cons of each ability

The second feature is a play on the classic matching memory game with Xcom flavour to it. The objective of the game is for the user to match the soldiers, whilst avoiding the aliens. It has the function of utilising the local storage to keep the user's high score so they can come back each time to try to beat it.

The game will also dynamically update the number of aliens in the card deck, depending on the difficulty that has been set. Give it a go and see how you get on!
